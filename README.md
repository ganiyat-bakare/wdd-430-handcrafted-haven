## Project Overview

Handcrafted Haven is a group project for WDD430. The application serves as a virtual marketplace for artisans and crafters to showcase and sell their unique handcrafted items. The platform connects talented creators with potential customers who appreciate the beauty and quality of handmade products. It provides a comprehensive solution for discovering, browsing, and purchasing authentic handmade products while fostering a sense of community, supporting local artisans, and promoting sustainable consumption.

## Technology Stack

### Frontend

- **Framework**: Next.js 15.0
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Library**: React 18

### Development Tools

- **Package Manager**: npm
- **Linting**: ESLint with Next.js configuration
- **Code Formatting**: Prettier (integrated with ESLint)
- **Version Control**: Git & GitHub

### Deployment

- **Platform**: Vercel
- **Domain**: Custom domain configuration
- **CI/CD**: Automated deployment from GitHub


### Prerequisites

You must have the following installed to start the project.

- Node.js 18.0 or higher
- npm or yarn package manager
- Git for version control

### Common Workflow Commands

You must have Node installed to run the following commands.

- `npm install` to install project dependencies.
- `npm run dev` to start up the local development server and updates on any JS or CSS/JSX.
- `npm run lint` to run ESLint against your code to find errors.
- `npm run format` to run Prettier to automatically format your code.
- `npm run start` starts up a local server and updates on any JS or CSS/SCSS.
- `npm run build` to build final files when you are ready to turn in.
